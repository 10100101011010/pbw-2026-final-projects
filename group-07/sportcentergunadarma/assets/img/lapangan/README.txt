taruh file gambar lapangan kamu (format .png atau .jpg) di folder ini.

setelah itu, buka panel admin > kelola lapangan, lalu isi kolom
"nama file gambar" dengan nama file PERSIS seperti yang kamu taruh
di sini, contoh: futsal-a.png

kalau kolom gambar dikosongkan atau nama filenya salah ketik,
kartu lapangan di halaman utama otomatis fallback ke kotak emoji
ungu placeholder, BUKAN gambar rusak/broken image icon.
