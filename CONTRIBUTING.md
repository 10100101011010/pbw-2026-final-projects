# Contributing Guide

## Folder Naming
Each group has a folder named `group-XX` (e.g. `group-01`, `group-02`, ...
`group-07`). Only add or edit files inside your own group's folder.

## Rules
- Do not edit other groups' folders.
- Do not edit the root README.md, .gitignore, or this file.
- Do not commit node_modules/, .env, or other secret/dependency files.
- Write clear commit messages describing what you changed.

## Workflow
1. `git pull`
2. Make your changes inside your group folder (e.g. `group-03/`)
3. `git add group-XX` (replace XX with your group number)
4. `git commit -m "Describe your change"`
5. `git push`
