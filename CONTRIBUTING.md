\# Contributing Guide



\## Folder Naming

Each group has a folder named `group-XX-projectname`. Only add or edit

files inside your own group's folder.



\## Rules

\- Do not edit other groups' folders.

\- Do not edit the root README.md, .gitignore, or this file.

\- Do not commit node\_modules/, .env, or other secret/dependency files.

\- Write clear commit messages describing what you changed.



\## Workflow

1\. `git pull`

2\. Make your changes inside your group folder

3\. `git add group-XX-yourprojectname`

4\. `git commit -m "Describe your change"`

5\. `git push`
