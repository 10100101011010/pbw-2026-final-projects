Group 03 - Findly Friendly

Findly Friendly adalah sistem pelaporan dan pencarian barang hilang atau barang temuan di lingkungan Kampus Universitas Gunadarma. Dibuat untuk tugas akhir mata kuliah Pemrograman Berbasis Web.

Tentang Project

Findly Friendly adalah platform berbasis website yang dirancang khusus untuk membantu civitas akademika atau mahasiswa dalam mengembalikan atau menemukan barang-barang yang tercecer di lingkungan kampus.
